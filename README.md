O desafio de uma empresa de tecnologia focada em serviços de TI é gerenciar o fluxo de trabalho (Workflow) desde a captação do cliente até a entrega do serviço e suporte pós-venda. O objetivo é criar um sistema que centralize a gestão de clientes, a alocação de colaboradores e o monitoramento do ciclo de vida de cada Chamado, garantindo a aplicação das regras de negócio e dos princípios de OO.
